# Éclore Before / After

A black-and-white Helvetica / Swiss-grid before-after presentation app for Éclore Aesthetics partner review.

## Local run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Static build

```bash
npm run build
```

For GitHub Pages export paths:

```bash
GITHUB_PAGES=true npm run build
```
